Company History
Short overview & hiring expectations
Founded: 1945 • Headquarters: Bengaluru, India

Wipro Limited is a leading Indian multinational corporation founded in 1945. Originally a vegetable oil company, it diversified into IT services in the 1980s and is now among the top global IT consulting and outsourcing companies. Wipro operates in over 60 countries, providing IT, consulting, and business process services.

What Wipro typically looks for in freshers:

  • Good knowledge of programming concepts and problem-solving skills.
  • Strong fundamentals in computer science (OOPs, DBMS, Networking).
  • Web basics knowledge (HTML, CSS, JavaScript) is a plus.
  • Communication skills, adaptability, and team collaboration.

HTML — Common questions
1. What is semantic HTML? HTML
Answer: Semantic HTML uses meaningful tags like <header>, <article>, <footer> that describe content clearly for both browsers and developers.
2. Difference between inline, block, and inline-block elements?
Answer: Inline does not start on a new line (span), block takes full width (div), inline-block allows block features but stays inline.
CSS — Common questions
3. Difference between relative, absolute, fixed positioning? CSS
Answer: Relative is offset from normal position, absolute is positioned relative to nearest positioned ancestor, fixed is positioned relative to viewport.
4. What are media queries in CSS?
Answer: Media queries allow responsive design by applying styles based on device width/height or other conditions.
JavaScript — Common questions
5. Explain hoisting in JavaScript. JS
Answer: Hoisting means variable and function declarations are moved to the top of their scope before execution. But let/const are hoisted in the “temporal dead zone.”
6. What is the difference between == and === ?
Answer: `==` checks equality after type coercion, while `===` checks both value and type.
HR Round — Common questions
7. Tell me about yourself.
Answer: Introduce education, projects, technical skills, and how they align with Wipro’s values and global projects.
8. Why do you want to join Wipro?
Answer: Mention Wipro’s global exposure, innovation culture, and opportunities to work on cutting-edge projects.
