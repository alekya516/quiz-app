Company History
Short overview & hiring expectations
Founded: 1968 • Headquarters: Mumbai, India

Tata Consultancy Services (TCS) is an Indian multinational information technology services and consulting company and part of the Tata group. Founded in 1968, TCS provides IT services, consulting, and business solutions to clients worldwide. Over decades it has grown into one of the world's largest IT employers and service providers.

What TCS typically looks for in campus / entry hires:

  • Strong fundamentals in programming (data structures, basics of algorithms).
  • Good understanding of web basics if applying for front-end or full-stack roles (HTML/CSS/JS).
  • Ability to solve logic/problem-solving questions, clear communication, and teamwork.
  • Professionalism: punctuality, willingness to learn, and adaptability to different projects.

Practice Mode
HTML — Common questions
1. What is the difference between <div> and <section> ? HTML
Answer: <div> is a generic container with no semantic meaning. <section> is semantic and represents a standalone section of content, often with a heading. Use semantic tags for accessibility and SEO.
2. What is the use of the alt attribute on <img>? Accessibility
Answer: The alt attribute provides alternative text for images for screen readers and when images fail to load. It improves accessibility and SEO.
3. How do you create a link that opens in a new tab? HTML
Answer: Use <a href="URL" target="_blank" rel="noopener noreferrer">Link</a>. `rel="noopener noreferrer"` improves security and performance.
CSS — Common questions
4. Difference between class and id selectors? CSS
Answer: `.` selects classes and can be reused multiple times. `#` selects an id which should be unique on a page. ID has higher specificity than class.
5. How does the CSS box model work? Layout
Answer: Box model contains content, padding, border, and margin. Width/height apply to content box by default. `box-sizing: border-box` includes padding and border within width/height.
6. What is Flexbox and when to use it?
Answer: Flexbox is a layout mode for distributing space along one axis (row/column). Use it for horizontal/vertical alignment, dynamic resizing, and simple responsive layouts.
JavaScript — Common questions
7. Difference between var, let and const? JS
Answer: `var` is function-scoped and hoisted; `let` and `const` are block-scoped. `const` cannot be reassigned (but objects can be mutated).
8. What is event delegation?
Answer: Event delegation uses a single event listener on a parent element to handle events from child elements (using event bubbling). It reduces listeners and improves performance.
9. How do promises work? What is async/await?
Answer: A Promise represents an async operation that can resolve or reject. `async/await` is syntactic sugar to write promise-based code in a synchronous-looking style: `await` pauses until the promise resolves.
HR Round — Common questions
10. Tell me about yourself.
Answer (sample): Give a short elevator pitch: your name, education, key projects/skills relevant to the role, and one strength + how you plan to grow at TCS. Keep it concise (60–90 seconds).
11. Why do you want to join TCS?
Answer (tip): Mention TCS's learning culture, large-scale projects exposure, and long-term career growth. Tie it to your goals and what you can contribute.
12. What are your strengths and weaknesses?
Answer (tip): Choose strengths that match the job (teamwork, learning quickly). For weakness, pick a real area you're improving and explain how you're addressing it.
Quick tips
